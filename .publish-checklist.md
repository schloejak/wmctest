## Publishing Steps

### 1. Build Library
```bash
npm run build:lib
```

### 2. Test Build Output
```bash
ls -la dist/
```

### 3. Login to npm (falls nötig)
```bash
npm login
```

### 4. Publish
```bash
npm publish --access public
```

### 5. Verify
```bash
npm view @yourusername/angular-auth-template version
```
